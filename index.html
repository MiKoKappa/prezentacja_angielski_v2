UX is the foundation to UI design.
You cannot start the second without the first one.

UX is the first step for creating a great product for the customer. You can't skip UX or change the order (if you aim for the success of course).

Why is it so important?

Why is this whole design trend so important?

It is all about companies' demand.

To gain and retain customers the product must be presented in the best possible way.

How many times you have searched for something, found a page and imediatelly quit it because of tons of popups, agreements and so on? Companies are starting to see that a frustrated client is the worst thing that can happen. Therefore they are spending more money on planning their product to better fit users preferences, to bring joy and satisfaction to people using it, because that is what brings cash.

What tools are available for UX/UI designers?

User research tools

I would like to split the tools into three groups, based on designing steps. The first one is user resarch tools. To better know what the user really wants and what he aims for you need to survey, interview and find value in gathered data.

User research tools

Dovetail - creating research notes based on interviews

Dovetail is an example of note taking app for UX researchers. It enables you to produce voice transciptions, annotate them, collaborate on notes with your whole team.

User research tools

Google Forms - collecting customer satisfaction data by forms

Next tool should be a well know one in our group. Google Forms is making creating and getting the results of online test forms an easy task. You can use it to question potential users about their needs for example.

User research tools

Google Analytics - finding patterns in collected data

After all the data is collected you need to get value from it. Google Analytics is a tool for aggregating, finding patterns and seeing what changes would bring the most positive impact on the product.

Design tools

The next group is design tools for when you know what you should do and you need to now make it graphically available.

Design tools

Lucidchart - flowcharting to visualize user journey

Lucidchart is an UML tool for creating flowcharts of user journey, showing steps or states in which a user can be and what actions he need to take on to continue to desired state.

Design tools

OmniGraffle - diagramming navigation in apps focusing on Apple devices

OmniGraffle is similar but pays more attention to graphical aspect of user experience. It enables to create flows of mobile app views, usefull for prototyping Apple iOS programs.

Design tools

Balsamiq - wireframing tool for quick UI prototype devices

Balsamiq is all about wireframing and transforming ideas into basic UI templates. You can use predefined blocks of app elements to prototype layouts.

Design tools

Figma - design and prototyping tool built with components in mind

Figma is the most complex tool in this group. When using it you can design whole UI view, split parts into components, export code for programmers, and prepare working prototype of app to show your client.

Satisfaction checking tools

The last group is satisfaction checking tools. After the app has been published you can track the usage and adjust minor things that aren't perfect.

Satisfaction checking tools

Hotjar Heatmaps - graphical representation of user interests on page

Hotjar Heatmaps produce graphical representation of what a statistical user does on your page or app. It tracks cliks, drags, scrolls and hovers to show UX/UI designer what brings more interest to the user.

Satisfaction checking tools

Lucky Orange - conversion rate optimization tool based on heatmaps and session recordings

Lucky Orange is very similar to hotjar heatmaps, but it focuses on business part - optimizing the conversion rate, mainly on online shopping sites.

Satisfaction checking tools

SmartLook - All-in-one app analytics tool

SmartLook is an all in one solution: analyzing user interaction by heatmaps and session recording, timelines of views and other usefull information sources.
